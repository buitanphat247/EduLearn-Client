# Tạo lớp học mới

Dành cho Giáo viên muốn tạo không gian học tập riêng.

## Các bước tạo lớp

1. Đăng nhập tài khoản Giáo viên.
2. Vào trang **Quản lý lớp học**.
3. Nhấn nút **+ Tạo lớp học**.
4. Điền thông tin:
   - Tên lớp học (Ví dụ: Toán 12 - Thầy Nam)
   - Mô tả ngắn
   - Ảnh bìa lớp học
5. Nhấn **Lưu**.

## Mời học sinh

Sau khi tạo lớp, bạn sẽ nhận được một **Mã lớp học** (Class Code). Hãy gửi mã này cho học sinh hoặc thêm trực tiếp họ vào lớp.
